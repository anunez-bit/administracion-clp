# Administración CLP

Versión estática independiente conectada al proyecto existente de Supabase.

## Publicar con GitHub Pages

1. Sube `index.html` y `.nojekyll` a la raíz del repositorio.
2. En GitHub abre **Settings → Pages**.
3. En **Build and deployment**, selecciona **Deploy from a branch**.
4. Elige la rama `main`, carpeta `/ (root)` y pulsa **Save**.
5. GitHub mostrará la dirección pública después de unos minutos.

La clave incluida es la clave pública/publishable de Supabase. La privacidad depende de las políticas RLS configuradas en Supabase.
